# Нет энергетикам

PWA-трекер дней без энергетиков. Открывается в Safari на iPhone, добавляется на экран
«Домой» и работает как нативное приложение — без App Store.

Сделано с теплом для конкретного человека: отметить день, видеть серию 🔥, вести заметки,
получать поддержку. ИИ-помощник на Qwen/Cerebras — следующим этапом.

## Возможности (этап 1 — готово)
- **Сегодня:** большая кнопка «без энергетика», огонёк-серия, статистика, мотивация.
- **Календарь:** месячный грид, цветные статусы, тап по любому дню → статус + заметка.
- **Награды:** ачивки за 1 / 3 / 7 / 14 / 30 / 60 / 100 дней.
- **Офлайн:** данные в IndexedDB; с настроенным Supabase — синхронизация в облако.

## Стек
React 19 · TypeScript · Vite · Tailwind v4 · vite-plugin-pwa · Zustand · Dexie · Supabase ·
Framer Motion · date-fns.

## Разработка
```bash
npm install
npm run dev          # http://localhost:5173 (и адрес LAN — для теста с iPhone)
npm run build        # сборка в dist/ (typecheck + Vite + service worker)
npm run preview      # предпросмотр прод-сборки
npm test             # юнит-тесты логики серии
```

### Тест на iPhone (локально)
1. `npm run dev` — Vite слушает и на LAN (видно адрес `http://192.168.x.x:5173`).
2. iPhone в той же Wi-Fi → открыть этот адрес в **Safari**.
3. Для проверки установки/офлайна (нужен HTTPS) — тестировать на деплое Vercel.

## Облако (Supabase)
Приложение работает и без облака (локальный кеш). Чтобы включить синхронизацию:
1. Создать проект на [supabase.com](https://supabase.com).
2. SQL для таблиц — см. [supabase/schema.sql](supabase/schema.sql).
3. Заполнить `.env` (см. `.env.example`).
