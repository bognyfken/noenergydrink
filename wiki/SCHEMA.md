**Last updated:** 2026-06-20

## Назначение
Постоянная база знаний проекта «Нет энергетикам» — PWA-трекер дней без энергетиков.
Ведётся ассистентом (Claude), курируется человеком. Живёт в `wiki/`, открывается в Obsidian
прямо на этой папке. Это **память между сессиями** — читать вместо повторного вывода контекста.

## Слои (паттерн Karpathy LLM Wiki)
1. **`sources/`** — сырьё (скриншоты, заметки, требования). **Gitignored**, локально. Только чтение.
2. **`pages/`** — страницы по темам. Ассистент владеет слоем целиком (создаёт/обновляет/мёрджит).
3. **`queries/`** — сохранённые разборы/анализы, которые стоит сохранить. В git.

### Файлы в корне `wiki/`
- `index.md` — каталог страниц по категориям
- `log.md` — append-only хронология операций над вики
- `SCHEMA.md` — этот файл
- `templates/Wiki Page.md` — шаблон новой страницы

## Воркфлоу
- **Query:** на вопрос — сперва `index.md`, потом нужные страницы; нетривиальный ответ
  сохранить в `queries/YYYY-MM-DD slug.md`, дописать в `log.md`.
- **Sync:** после значимого изменения кода — обновить затронутые страницы (+ timestamp),
  дописать `UPDATED` в `log.md` (с хешем коммита, если есть), подвинуть пункты в `Roadmap`.
- **Lint:** периодически — противоречия, устаревшее, осиротевшие страницы, пробелы.

## Формат страницы
```markdown
# Заголовок

**Last updated:** YYYY-MM-DD
**Sources:** [[источник]]

---

## Раздел
Текст с [[ссылками]].

## See also
- [[Связанное]]
```

## Формат log.md
- Заголовок по дате: `## YYYY-MM-DD`
- Пункты: `- ACTION: краткое описание` (со `[[ссылками]]`)
- Actions: `INIT` | `CREATED` | `UPDATED` | `DECISION` | `INCIDENT` | `LESSON` | `RESEARCH`
- Append-only — прошлые записи не переписываем.
