# Промты Nano Banana для арта ачивок

**Last updated:** 2026-06-20
**Sources:** [[Design]] · `src/lib/achievements.ts` · `public/achievements/*.webp`

---

Промты для генерации бейджей наград в **Nano Banana** (Gemini image). Стиль задан по
существующим монетам `d1` (огонёк), `d30` (веточка лаванды), `d100` (золотая корона).
Всего наград 37 (см. [[Features]] · [[Data Model]]). Уже есть арт: `d1 d3 d7 d14 d30 d60 d100`.

## Как пользоваться
1. Скопируй **БАЗОВЫЙ СТИЛЬ**.
2. Добавь снизу строку нужного бейджа (**Кант / Центр / Акцент**).
3. Сохрани результат как `public/achievements/<id>.webp` (квадрат, прозрачный/белый фон),
   затем выставь `hasImage: true` у этой ачивки в `src/lib/achievements.ts`.

## БАЗОВЫЙ СТИЛЬ (префикс к каждому)
```
Иконка-награда для мобильного приложения в виде круглой медали-монеты. Мягкий 3D-стиль
матовой эмали и пластилина (claymorphism): объёмный скруглённый кант, по центру слегка
вдавленный/выпуклый символ. Мягкий студийный свет, нежный блик сверху, лёгкая тень снизу.
Объект строго по центру, на прозрачном или чистом белом фоне. Без текста, без букв, без
надписей, без водяных знаков, не фотореализм. Единый аккуратный премиальный набор иконок.
Палитра: фиолетовый #7c5cbf, лаванда #cdb4f6, глубокий тёмно-фиолетовый #160f22, тёплое
золото-пламя #f0a35e, мягкая роза #c46b8a. Квадрат 1:1, высокая детализация.
```
Негатив (если просит поле): `текст, буквы, цифры-надписи, люди (кроме указанных), реализм-фото, рамки с текстом, водяной знак`.

## Кант — три пресета
- **Лавандовый** (светлая монета `#cdb4f6`) — ранние/обычные награды.
- **Глубокий фиолетовый** (`#7c5cbf`) — средние.
- **Тёмный + золото** (тёмно-фиолетовая монета `#160f22`, золотой символ, мягкое свечение) — топовые и секретные.

---

## Серия подряд (streak) — мотив огня, растёт
- **d1 · Первый день** — Кант: лавандовый. Центр: один маленький тёплый огонёк-уголёк. Акцент: золото-пламя. *(есть)*
- **d3 · Три дня** — Кант: лавандовый. Центр: огонёк с тремя искорками вокруг. Акцент: золото. *(есть)*
- **d7 · Неделя** — Кант: лавандовый. Центр: огонёк в кольце из семи точек. Акцент: золото. *(есть)*
- **d14 · Две недели** — Кант: лавандовый. Центр: пламя побольше, две искры-дуги. Акцент: золото. *(есть)*
- **d21 · Три недели** — Кант: глубокий фиолетовый. Центр: росток-сеянец с крошечным огоньком-бутоном (привычка прорастает). Акцент: зелёно-золотой.
- **d30 · Месяц** — Кант: лавандовый. Центр: веточка лаванды. Акцент: лаванда. *(есть)*
- **d60 · Два месяца** — Кант: глубокий фиолетовый. Центр: два сплетённых пламени / устойчивый костёр. Акцент: золото.
- **d90 · Три месяца** — Кант: глубокий фиолетовый с лёгким золотом. Центр: высокое уверенное пламя. Акцент: золото.
- **d100 · 100 дней** — Кант: тёмный + золото. Центр: золотая корона. Акцент: золото. *(есть)*
- **d180 · Полгода** — Кант: тёмный + золото. Центр: золотой лавровый венок вокруг пламени. Акцент: золото, мягкое свечение.
- **d365 · Целый год** — Кант: тёмный royal-фиолет + золото. Центр: золотое солнце-кольцо с пламенем в центре, самый нарядный. Акцент: сияющее золото.

## Всего чистых дней (total) — мотив накопления
- **total7 · Первая семёрка** — Кант: лавандовый. Центр: семь маленьких звёздочек, собранных горкой. Акцент: золото.
- **total30 · 30 чистых** — Кант: лавандовый. Центр: горсть мелких самоцветов-капель. Акцент: лаванда.
- **total50 · 50 чистых** — Кант: глубокий фиолетовый. Центр: аккуратная стопка монеток. Акцент: мягкое золото.
- **total100 · 100 чистых** — Кант: глубокий фиолетовый. Центр: банка-баночка, наполненная светящимися точками. Акцент: золото.
- **total200 · 200 чистых** — Кант: тёмный + золото. Центр: переполненная банка светящихся капель света. Акцент: золото.
- **total365 · Год чистых дней** — Кант: тёмный + золото. Центр: полное кольцо из множества крошечных точек, образующих сияющий круг. Акцент: золото.

## Возвращение (comeback) — мотив сердца и подъёма
- **comeback · Возвращение** — Кант: лавандовый с тёплой розой. Центр: мягкое сердце с маленькой стрелкой вверх. Акцент: роза `#c46b8a`.
- **comeback3 · Несгибаемая** — Кант: глубокий фиолетовый. Центр: крепкое сердце с тремя восходящими шевронами. Акцент: роза + золото.

## Календарь (calendar) — мотив дат
- **weekend1 · Чистые выходные** — Кант: лавандовый. Центр: листок календаря с двумя подсвеченными днями и крошечным солнцем. Акцент: золото.
- **weekend4 · Месяц выходных** — Кант: глубокий фиолетовый. Центр: календарь с четырьмя подсвеченными парами дней. Акцент: золото.
- **week_cal · Идеальная неделя** — Кант: лавандовый. Центр: ряд из 7 ячеек-дней, все светятся/с галочками. Акцент: лаванда.
- **month_cal · Идеальный месяц** — Кант: тёмный + золото. Центр: полная сетка месяца, все ячейки заполнены, в центре золотая звезда. Акцент: золото.
- **month20 · Почти идеальный месяц** — Кант: глубокий фиолетовый. Центр: сетка месяца, почти вся заполнена, пара пустых ячеек. Акцент: лаванда.
- **monday4 · Лёгкий понедельник** — Кант: лавандовый. Центр: ячейка-понедельник с маленьким рассветом/солнцем. Акцент: золото-пламя.
- **seasons · Времена меняются** — Кант: лавандовый. Центр: три маленьких листочка разных сезонных оттенков в ряд. Акцент: зелёно-золотой.
- **newyear · Новый год — новая я** — Кант: тёмный + золото. Центр: листок календаря «1 января» с крошечным фейерверком и снежинкой. Акцент: золото.

## Дисциплина учёта (logging) — мотив отметок
- **log7 · Внимательная неделя** — Кант: лавандовый. Центр: чек-лист с семью галочками. Акцент: мягкий бирюзово-лавандовый.
- **log30 · Дисциплина** — Кант: глубокий фиолетовый. Центр: плотная сетка-трекер привычки, вся в галочках. Акцент: бирюзовый.
- **marked50 · Полсотни отметок** — Кант: глубокий фиолетовый. Центр: стопка карточек с галочками. Акцент: лаванда.

## Заметки (notes) — мотив дневника
- **note1 · Первая мысль** — Кант: лавандовый. Центр: маленький блокнот с карандашом и сердечком. Акцент: лаванда.
- **note10 · Дневник чувств** — Кант: глубокий фиолетовый. Центр: открытый дневник с сердцем и строчками. Акцент: лаванда + роза.

## Секретные (secret) — тёмные, с мягким свечением, ощущение «особенного»
- **leapday · Високосный** — Кант: тёмный ночной фиолет + свечение. Центр: ячейка календаря «29» с крошечной прыгающей фигуркой и блёстками. Акцент: золотые искры.
- **friday13 · Пятница, 13** — Кант: полуночный фиолет + мягкое свечение. Центр: ячейка «13» с милым чёрным котиком/подковой и подмигиванием (не страшно, по-доброму). Акцент: золото.
- **night_owl · Полуночница** — Кант: тёмный сине-фиолетовый + свечение. Центр: милая сонная совушка под полумесяцем и звёздами. Акцент: серебристо-золотой свет.
- **second_wind · Второе дыхание** — Кант: тёмный + золото, свечение. Центр: мягкий вихрь ветра, вновь раздувающий пламя. Акцент: золото.
- **phoenix · Феникс** — Кант: тёмный royal-фиолет + сияние. Центр: маленький феникс, восстающий из мягких углей, тёпло-золотое пламя. Акцент: оранжево-золотое свечение.
