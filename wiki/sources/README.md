# sources/

Сырьё, на основе которого пишутся `pages/`. **Не в git** — только локально.

## Что сюда класть
- Скриншоты, заметки, требования, расшифровки.
- Любое личное/чувствительное.

## Соглашения
- Имя файла: `YYYY-MM-DD slug.ext`.
- При переносе в `pages/` ссылаться на источник.
- Не редактировать уже добавленные источники — иммутабельный слой.
