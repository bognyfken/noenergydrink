Нижний таб-бар: **Сегодня · Календарь · Награды · Помощник**.

## Сегодня (`screens/TodayScreen.tsx`)
- Шапка: лого (`public/logo.png`) + название + шестерёнка настроек.
- `StreakFlame` — огонёк с **числом серии ВНУТРИ** пламени (два слоя + мерцание); интенсивность/цвет
  растут с серией. Блок из трёх цифр убран; рекорд — в Наградах, «всего чистых» — в Календаре.
- `PrimaryButton` — «Сегодня без энергетика» (+ ссылка «Сорвалась сегодня»). День реактивен (`useToday` — сброс в 00:00).
- `MotivationCard` — мотивация на сегодня от **Кабанёнка** (`warm.motivation`: генерится и пред-генерится
  наперёд, всегда новая, меняется при заходе); запасная фраза — `lib/motivation.ts` (офлайн/пока не сгенерилось).
- `GreetingCard` — приветствие Кабанёнка при открытии (см. раздел «Помощник»).

## Настройки (шит профиля, в `TodayScreen`)
- Профиль: имя + «Сменить профиль» (выход → лок-экран/выбор профиля).
- `GreetingToggle` — **«Кабанёнок встречает»** (по умолчанию вкл): показывать ли приветствие при открытии.
- `ReminderToggle` (`components/ReminderToggle.tsx`) — **ежедневные пуш-напоминания** «отметь день».
  Web Push (`lib/push.ts` + `public/push-sw.js`), подписка в Supabase, рассылка — Vercel Cron 20:00 МСК.
  Состояния: вкл/выкл · «добавь на экран Домой» (iOS, не установлено) · «уведомления запрещены». См. [[Architecture]].

## Календарь (`screens/CalendarScreen.tsx`)
- `MonthGrid` — месяц, недели с понедельника, цвета: clean=фиолет, drank=розовый, unmarked=тёмный;
  сегодня — кольцо; точка-индикатор на днях с заметкой.
- Тап по дню → `DayEditorSheet` (нижний шит): статус + заметка + «очистить день». Любой день, в т.ч. прошлый.

## Награды (`screens/AchievementsScreen.tsx`)
- **37 ачивок** (`lib/achievements.ts`), 6 категорий + секретные. Разнообразные, не только по серии:
  - **streak** (11): 1/3/7/14/21/30/60/90/100/180/365 дней подряд.
  - **total** (6): 7/30/50/100/200/365 чистых дней в сумме.
  - **comeback** (2): возвращение после срыва · 3 возвращения.
  - **calendar** (8): чистые выходные · 4 уикенда · идеальная неделя пн–вс · идеальный месяц ·
    20 дней за месяц · 4 чистых понедельника · 3 разных месяца · чистое 1 января.
  - **logging** (3): учёт 7/30 дней подряд · 50 отметок.
  - **notes** (2): первая заметка · 10 заметок.
  - **секретные** (5, скрыты как «???»): Високосный · Пятница 13 · Полуночница · Второе дыхание · Феникс.
- Каждая ачивка — предикат `test(stats)` над `AchStats` (см. [[Data Model]]).
- Дисплей **временно на эмодзи** (карта `EMOJI` по id в `AchievementsScreen`), пока не готовы арт-медали:
  закрытые — приглушены (`opacity-40 grayscale`) + замок; секрет до открытия — «❓».
- Арт-бейджи: `public/achievements/<id>.webp`; у всех 37 ачивок `hasImage: true` (готово под арт, есть
  `d1 d3 d7 d14 d30 d60 d100`, остальные 30 генерим в Nano Banana). Промты —
  [[2026-06-20 промты Nano Banana для бейджей]]. ⚠️ при возврате к картинкам недостающие `webp` дадут 404,
  пока живёт эмодзи-вариант.
- `AchievementToast` — всплывающий тост при новой ачивке (из стора, `justUnlocked`); для секрета — «Секретная награда» + ✨.

## Помощник — Кабанёнок (`screens/ChatScreen.tsx`)
Тёплый ИИ-собеседник на **Cerebras** (модель `zai-glm-4.7`; Qwen на аккаунте недоступен). Персона —
`lib/ai.ts` (`PERSONA`): представляется «Кабанёнком», НИКОГДА не зовёт себя ИИ/моделью; мягкий, на «ты»,
женские местоимения, не осуждает срывы (семейная манера кабан/кабаниха/кабанёнок).
- **Чат**: пузыри, ввод, индикатор «печатает», мягкие ошибки. Сам действует инструментами по ходу
  разговора: `set_day_status` / `add_day_note` (пишет дни и заметки) + плашка **«отменить»**;
  `remember` (долговременная память).
- **История бесед** (`components/ConversationsSheet.tsx`): список бесед, новый чат, переключение,
  переименование, удаление. Заголовок беседы — по первому сообщению; модели уходит история только
  активной беседы. Шапка чата: список слева, заголовок, «новый чат» справа.
- **Приветствие** (`components/today/GreetingCard.tsx`): при открытии «Сегодня», если день не отмечен
  и тумблер вкл — карточка с тёплым вопросом + 3 сгенерированных быстрых ответа + поле ввода → переброс
  в «Помощник» с авто-отправкой реплики (`location.state.send`).
- **Память + контекст**: память об Олесе (`lib/memory.ts`) и состояние (серия, статус, последние дни)
  инъектятся в системный промпт — Кабанёнок помнит и продолжает («вчера было тяжело — как сейчас?»).
- **Скорость**: GLM — «думающая» модель; чтобы опрос не тормозил, второй round-trip после tool-call
  делается только если модель не дала текст сразу (GLM часто отдаёт ответ вместе с вызовом инструмента).
- Прокси ключа — Cloudflare Pages Function `functions/api/chat.ts`. См. [[Architecture]] · [[Data Model]].
